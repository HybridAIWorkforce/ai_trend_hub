# AI Trend Hub - Project Instructions

## Owner
Jack Whatley - Author of "Reshaping Recruitment", AI & Business Transformation Strategist

## Project Overview
Personal AI trend tracking dashboard + content generation engine. Scrapes AI trends from RSS/Reddit, generates articles in Jack's authentic voice using his IP frameworks, with AIDA engagement and CTA monetization.

## Tech Stack
Next.js 14 (App Router), TypeScript, TailwindCSS, PostgreSQL, Prisma, NextAuth.js (JWT), Abacus.AI LLM (gpt-4.1-mini)

## Design Language
- Dark theme, almost black background with subtle gradients
- Card-based layout with rounded corners and thin borders
- Clean, minimal typography with soft glow effects
- Premium, futuristic, calm aesthetic

## Voice Profile
Jack's voice is conversational, direct, uses metaphors and stories. Opens with "Picture this..." or personal anecdotes. Signature phrases: "AI Never Sleeps", "Levels the playing field". Target audience: regular people, small business owners (NOT academics/PhDs). Core message: Hybrid AI Workforce (AI + Humans > AI alone).

## 7 IP Frameworks
1. Skyscraper Foundation - Build deep before tall (data foundation)
2. 6-Stage Recruitment Cycle - Documented system
3. 4 Hire Up Funnels - FOMO, Not Qualified Yet, New Activation, Abandoned
4. Co-Pilot System - AI accountability monitoring
5. Improvement Flywheel - Continuous optimization loop
6. Maslow-Based Messaging - Hierarchy of needs messaging
7. Hybrid AI Workforce - Collaboration NOT replacement

## Cross-Industry Support
Frameworks apply to: recruitment, sales/B2B, e-commerce, restaurants, healthcare, manufacturing, professional services, SaaS

## Key Features
- Auto-login as owner@aitrendhub.com / trendhub2024
- AIDA framework with guardrails (optional toggle)
- CTA management (book, video, affiliate products) in Settings
- Article wizard: Basics → Angles → Outline → Draft
- Cross-industry IP detection and injection
- 30-min background refresh for RSS/Reddit

## CTA Analytics
- CTAEvent model tracks impressions, clicks, copies per CTA
- GET /api/cta-analytics returns 30-day summary with per-CTA breakdowns
- UI in Settings → CTA Performance section with toggleable analytics panel

## Industry Detection (v2)
- 11 industries with weighted keyword scoring (boost 1-3x per term)
- Industries: recruitment, sales/B2B, e-commerce, restaurants, healthcare, manufacturing, professional services, SaaS, education, fintech, real estate
- Primary = highest-scoring non-recruitment industry
- Cross-industry prompts when 3+ industries detected
- Secondary industry context for broader appeal

## Social Media Posts (v2)
- Platform toggles (select which platforms to generate for)
- Edit-in-place with live character count
- Three-tier char warnings: green (<90%), yellow (90-100%), red (over)
- Smart truncation at sentence boundaries
- Suggested hashtags with click-to-swap functionality
- Topic extraction for context-aware hashtag generation
